🪖 Sistema de Registro de Personal - Fuerzas Militares
​Este proyecto consiste en una plataforma digital diseñada para la captura de datos, fotografía y firma del personal perteneciente a las Fuerzas Militares de Colombia.
​📋 Funciones Principales
​Identificación Oficial: Interfaz institucional con los colores y logos de las Fuerzas Militares.
​Captura Multimedia: Acceso integrado a la cámara y galería para la foto del personal.
​Firma Digital: Panel táctil para la recolección de firmas en tiempo real.
​Base de Datos: Estructura preparada para el registro de nombres, cédula, grado y unidad militar.
